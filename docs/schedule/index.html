How to subscribe (Apple / Google / Outlook)

📱 iPhone · Apple Calendar

  1. Tap the green Subscribe button above
  2. Confirm in the popup → Add
  3. Important: open Settings → Calendar → Accounts → Subscribed Calendars → ohanaclubs and set Fetch to Daily (Apple's default is weekly, so changes won't show up otherwise!)

💻 Mac · Apple Calendar

  1. Click the green Subscribe button
  2. In the dialog, set Auto-refresh to Every day

🤖 Android / Google Calendar

  1. Tap Copy URL (in the … menu)
  2. On a desktop browser, open Google Calendar → Add by URL
  3. Paste the URL, click Add Calendar. Syncs to all your devices.

Google refreshes subscribed calendars about once a day; you can't change it.

📧 Outlook

  1. Tap Copy URL
  2. In Outlook: Add calendar → Subscribe from web → paste the URL
