Calendar URL:
  • iPhone · Apple Calendar: tap Subscribe → Subscribe → Add. Set Fetch to Daily (default is weekly) in Settings → Calendar → Accounts → Subscribed Calendars.
  • Mac · Apple Calendar: click Subscribe; in the dialog set Auto-refresh to "Every day".
  • Android / Google Calendar: tap Copy URL, then on a desktop browser open Google Calendar → Add by URL, paste, click Add.
  • Outlook: Add calendar → Subscribe from web → paste the URL.
